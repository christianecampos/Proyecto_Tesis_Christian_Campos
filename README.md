# Proyecto_Tesis_Christian_Campos

Repositorio de trabajo para el desarrollo de la tesis de Economía Aplicada (UBA - MAE).

Autor: Christian Campos

## Descripción inicial

Este repositorio contiene los archivos, scripts y documentación necesarios para el procesamiento y análisis de datos utilizados en la tesis.

Incluye:

- Datos trimestrales (ENAHO 2025)
- Procesamiento y unificación de archivos CSV
- Conversión a formato Parquet
- Construcción de base anual
- Análisis econométrico posterior
